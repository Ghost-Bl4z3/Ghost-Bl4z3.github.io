Source Code Analyzer

A Python-based web recon tool that tears apart HTML source code to surface security-relevant data — the kind of intel that matters in the early stages of a pentest or bug bounty.

  • Extracts URLs, JS files, IP addresses & emails
  • Discovers hidden/interesting endpoints via wordlists
  • Supports internal wordlist or custom wordlist modes
  • Designed for authorized security testing & recon
